3. **ProgressView Warning**: Ignored harmless SwiftUI floating-point precision warning in console

### Technical Notes

- `NSApplication.shared.setActivationPolicy(.accessory)` - menu bar only, no dock icon
- `NSApplication.shared.setActivationPolicy(.regular)` - shows dock icon
- FDA path for development builds: `~/Library/Developer/Xcode/DerivedData/MacSweep-xxx/Build/Products/Debug/MacSweep.app`
- Window close detection uses `NSWindow.willCloseNotification` with delayed check for remaining windows
